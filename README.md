# RuneTech

RuneTech is a static digital agency website for web design, development, automation, portfolio work, case studies, pricing, blog content, and lead generation.

## Pages

- `index.html` — Home page with agency hero, services, featured projects, process, pricing preview, and lead form.
- `services.html` — Full service overview.
- `portfolio.html` — Project portfolio with category filters.
- `case-studies.html` — Detailed project story examples.
- `pricing.html` — Service packages, add-ons, and FAQ.
- `blog.html` — Static agency blog overview.
- `contact.html` — Dedicated contact and project inquiry page.
- `styles.css` — Shared responsive styling.
- `script.js` — Smooth scrolling, project filtering, form messages, and footer year.

## How to view locally

Open `index.html` directly in your browser.

Optional local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Customizing content

Update the HTML pages directly to edit services, portfolio projects, case studies, pricing packages, blog cards, and contact details.

## Lead forms

The forms are front-end demos. They show a success message, but they do not send email or save submissions until connected to a form service or backend.

Common options for a real form connection include Formspree, Netlify Forms, a CRM form embed, or a custom backend endpoint.
